# 设计规范 v3 — 2026年世界杯·完整赛程指南

## 设计语言：Apple 纯净极简

### 设计原则
1. **内容优先** — 赛程信息是第一优先级，去除多余装饰
2. **时间线叙事** — 用日期分割线引导用户从上到下浏览
3. **沉浸式阅读** — 大量留白，清晰的字体层级
4. **细腻反馈** — 点击上浮、hover蓝框、active微缩

### 色彩系统
- 背景：纯白 `#FFFFFF`
- 次级背景：`#F5F5F7`
- 主文字：`#1D1D1F`
- 次级文字：`#6E6E73`
- 强调色：Apple Blue `#0066CC`
- 状态色：红(进行中) `#FF3B30` / 绿(已结束) `#248A3D` / 灰(未开始) `#8E8E93`
- 深色模式自动适配

### 字体
- SF Pro Display / SF Pro Text + PingFang SC
- 字号层级：11px(标签) → 13px(次级) → 14px(队名) → 22px(比分) → 32-60px(标题)

### 组件
- **导航栏**：毛玻璃效果，分段选择器筛选
- **Hero区**：轻柔渐变背景，毛玻璃倒计时，统计数据
- **日期分隔线**：`6月12日 ────` 样式
- **比赛卡片**：白底灰边框，hover变蓝 + 阴影上浮
- **底部弹窗**：iOS Sheet风格，从底部滑入，显示球队详情
