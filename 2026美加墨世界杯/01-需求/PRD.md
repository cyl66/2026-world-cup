# PRD v2: 2026年美加墨世界杯观赛指南网页

## 1. 产品概述
- **产品名称**：2026年世界杯 · 完整赛程指南
- **目标用户**：中国足球爱好者
- **核心价值**：基于真实赛程数据，按时间排序展示全部104场比赛，支持球队下钻
- **数据来源**：Excel表格 `2026年美加墨世界杯赛程表_豆包AI生成.xlsx`
- **平台**：单页HTML，移动端适配

## 2. 用户画像
- **核心用户**：中国足球迷，Apple设备用户，追求极简设计
- **使用场景**：在手机上快速查看今天/明天有什么比赛

## 3. 功能需求（更新版）

### P0 - 核心功能（必须）
1. ✅ **104场完整赛程**：基于Excel真实数据，包含72场小组赛+32场淘汰赛
2. ✅ **按时间排序**：严格按开赛时间，最早的在最上面，自上而下
3. ✅ **比赛状态**：实时显示"未开始/🔴进行中/已结束"，已结束自动生成模拟比分
4. ✅ **球队下钻**：点击卡片查看两队详情，再下钻查看球队完整介绍
5. ✅ **全部中文**：球队名、场地、轮次、描述全部用中文

### P1 - 筛选功能
6. ✅ **按阶段筛选**：全部赛程/小组赛/淘汰赛

### P2 - 体验增强
7. ✅ **倒计时**：距首场比赛实时倒计时
8. ✅ **深色模式**：跟随系统自动切换
9. ✅ **响应式适配**：手机/平板/桌面

## 4. 数据规格（来自Excel）
- 48支参赛球队，12个小组(A-L)，每组4队
- 小组赛：12组×6场=72场，每组3轮
- 淘汰赛：1/16决赛16场→1/8决赛8场→1/4决赛4场→半决赛2场→三四名决赛1场→决赛1场
- 比赛场馆：16个（美国10个、加拿大2个、墨西哥3个）
- 时间：全部为北京时间

## 5. 设计约束
- Apple极简风格（SF字体、毛玻璃、大量留白）
- 全部中文呈现
- 单页HTML，离线可用
- 按日期时间轴分组
