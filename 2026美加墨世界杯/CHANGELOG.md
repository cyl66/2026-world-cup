# 变更日志 (Changelog)

## v4.0 (2026-06-04 00:11)
> 最终稳定版 — 全功能交付

### 新增
- 移动端4个统计项（48参赛球队/12小组/104比赛场次/16场馆）单行显示，`flex-wrap:nowrap`
- HTTP服务器添加 `Cache-Control: no-cache` 头，防止浏览器缓存旧版本
- HTML头部添加 `<meta>` 缓存控制标签

### 修复
- **外网与本地不一致**：根因是 HTTP 服务器默认路由 `/` 返回旧的 `index.html`，
  而新文件是 `2026-world-cup.html`。修复：用最新文件覆盖 `index.html`
- **移动端赛程卡片上下错开**：删除 `flex-direction:column`，改为水平排列

## v3.0 (2026-06-03 23:54)
> Hero重设计 + 国家详情修复

### 新增
- **国家详情功能**：`bindCty()` 函数绑定"查看完整介绍"按钮点击事件
- 外网部署：`localhost.run` SSH 隧道，`https://*.lhr.life`

### 设计变更
- **导航栏**：白色毛玻璃 → 深色毛玻璃 `rgba(15,12,41,0.85)` + 金色底边 `rgba(255,215,0,0.15)`
- **Logo**：黑色 → 白色文字 + 金色图标
- **Hero背景**：浅灰白渐变 → 深色奢华渐变 `#0f0c29 → #1a1040 → #0a1628`
- **标题**：纯黑 → 金色渐变 `#ffffff → #ffd700` + 发光效果 `drop-shadow`
- **统计卡片**：普通文字 → 毛玻璃面板 `backdrop-filter:blur(12px)` + 金色数字
- **背景图案**：无 → 六边形网格纹路 `opacity:0.03`
- **底部分割线**：无 → 金色渐变分割线
- **徽章**：浅蓝背景 → 深色透明 + 金色边框
- **倒计时**：白色背景 → 深色毛玻璃
- **深色模式适配**：各组件完善

### 技术重构
- **事件委托替代内联 onclick**：`data-t1`/`data-t2` 属性 + `closest('.m')`，
  彻底避免 Python 生成 HTML 时的引号转义问题
- 修复 Python 原始字符串 `r'''` 中 `\"` 输出为 `"` 的问题
- JS字符串全部改用单引号，HTML属性使用双引号，互不干扰
